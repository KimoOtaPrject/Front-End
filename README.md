KimoOta Project 입니다.

## 시작하기

### 1. 개발모드 에서 서버 시작
```bash
yarn dev
```
>[`http://localhost:3080`](http://localhost:3080) 또는 [`https://localhost:3443`](https://localhost:3443) 에서 시작

### 2. 배포하기
```bash
yarn build && yarn start
```

### 3. 사용한 기술 스택
`React` `TypeScript` `Sass` `NextJS` `redux` `Ant.Design`
